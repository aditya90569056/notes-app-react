# 📝 Notes App (React + Vite)

A simple and fast Notes Application built using **React** and **Vite**. This app allows users to create, manage, and delete notes efficiently with a clean and responsive UI.

---

## 🚀 Features

* ✍️ Create new notes
* 🗑️ Delete notes
* 📋 View all saved notes
* ⚡ Fast performance with Vite
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Build Tool:** Vite
* **Styling:** Tailwind 
* **State Management:** useState

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone git clone https://github.com/aditya90569056/notes-app-react
```

2. Navigate to the project folder:

```bash
cd notes-app-react
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```
